# Project Creator Flow (Backoffice dApp)

#### **4.1 Overview**

What project creators do and why.

#### **4.2 Step-by-Step Flow**

1. Create escrow
2. Define milestones
3. Set approver + release signer
4. Deploy token
5. Deploy token sale contract
6. Launch sale page

#### **4.3 UI Components Used**

* Milestone editor
* Token metadata form
* Token sale parameters
* Deploy buttons

#### **4.4 API & SDK References**

* createEscrow
* createToken
* createTokenSale
* escrow roles
